# Digital Clock App

A sleek, modern digital clock built with React and Vite. This application displays the current time with a clean user interface and smooth updates every second.

## Features

- **Real-time Clock**: Displays current time including hours, minutes, and seconds.
- **AM/PM Support**: Automatically handles meridiem formatting.
- **Padded Time**: Ensures a consistent look with zero-padding for single-digit values.
- **Responsive Design**: Looks great on various screen sizes.

## Technologies Used

- **React**: For the user interface and state management.
- **Vite**: For a fast development environment and build process.
- **CSS**: For modern styling and layout.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository (if applicable) or download the source code.
2. Navigate to the project directory:
   ```bash
   cd my-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

To start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:5173`.

## Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles the project for production.
- `npm run lint`: Runs ESLint to check for code quality issues.
- `npm run preview`: Previews the production build locally.
